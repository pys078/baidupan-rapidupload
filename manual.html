常见问题

1. 如何转存秒传链接？

填入 秒传链接, 转存路径, 点击'解析链接并转存',
转存后弹窗显示的"errno"(错误码)即转存结果:
{"errno":0} 为转存成功, 其余转存失败的errno请见参考文档

2. 如何生成秒传链接？

选择文件后，等待秒传链接出现。
大文件计算速度较慢，请耐心等待。
需要任意账号的网盘中存有至少一份文件才能保证秒传链接的有效性!

工具说明

原作者: TkzcM、mengzonefire，该版本作者pys078

此工具为纯静态页面, 不会发送任何数据到网站服务器, 源码: Github

此工具支持 url 传参(一键秒传), 格式：[网站域名]/#bdlink=[参数], [参数]为 base64 加密过的任意格式链接 (支持批量)

若有任何问题请前往 Github 反馈
